# SpaceX

A simple api project using svelte and tailwind css



## Installation

Install my-project with npm

```bash
  npm install spaceX-main
  cd spaceX-main
```

## Development
Run project in development mode
```bash
npm run dev
```


## Deployment

To deploy this project run

```bash
npm run build
```
